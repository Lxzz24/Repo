## ② 求数值型数组中元素的最大值、最小值、平均数、总和等

数组：10~99的随机数

  ```java
    int[] arr = new int[10];
    for (int i = 0; i < arr.length; i++) {
        arr[i] = (int) (Math.random() * 90 + 10);
    }
  ```

### 求数组元素的最大值

  ```java
    int maxValue = arr[0];
    for (int i = 1; i < arr.length; i++) {
        if (maxValue < arr[i]) {
            maxValue = arr[i];
        }
    }
  ```

### 求数组元素的最小值

  ```java
    int minValue = arr[0];
    for (int i = 1; i < arr.length; i++) {
        if (minValue > arr[i]) {
            minValue = arr[i];
        }
    }
  ```

### 求数组元素的总和

  ```java
    int sum = 0;
    for (int i = 0; i < arr.length; i++) {
        sum += arr[i];
    }
  ```

### 求数组元素的平均值

`double avgValue = sum / arr.length;`
