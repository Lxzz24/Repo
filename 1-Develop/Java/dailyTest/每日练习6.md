# 1.使用冒泡排序，实现如下的数组从小到大排序。
` int[] arr = new int[]{34,5,22,-98,6,-76,0,-3};`
```java
int[] arr = new int[]{34,5,22,-98,6,-76,0,-3};

for(int i = 0;i < arr.length - 1;i++){
	for(int j = 0;j < arr.length - 1 - i;j++){
		if(arr[j] > arr[j + 1]){
			int temp = arr[j];
			arr[j] = arr[j + 1];
			arr[j + 1] = temp;
		}
	}
}
```
> 快排时间复杂度：O(nlogn)
> 冒泡时间复杂度：O(n^2)
> 堆排序、归并排序
